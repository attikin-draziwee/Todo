# ToDo application

We create a TODO application on Rust + Axum as a main web framework.

## Techonogyes
- Tokio as asynchronous runtime
- AXUM (by Tokio) as web application framework

## Features for this project:
- CRUD (Create-Read-Update-Delete) operations on all the TODO's 
- Persistence using database
- Authentication (JWT based)
- Monitoring (logging)
